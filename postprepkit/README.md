# Post Prep Kit Sent Scripts
These scripts are triggered after a prep kit has been sent to translation. This may be used to start some other process via a REST API for instance, or to gather / report data for custom uses, etc.

# Environment Variables
The following environment variables are available from these scripts:
*  LRM_GROUP
* LRM_PROJECT
* KIT_VERSION 
* LRM_LOCALE 
* PREPKIT_STATUS
